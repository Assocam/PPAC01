
#Qui elenchiamo le caratteritiche delle API che sviluppiamo

CONVENZIONI
- TABELLE e CAMPI in maiscolo
- parametri in minuscolo
- Classi in PascalConvention

Endpoint      :/login
Methods       : POST
Input         : json - username, password
Example       : {"username":"pippo", "password": "123"}       
Output        : Json  - result
Example       : {"result": "login OK"}, 200
                {"result": "not found"}, 404


Endpoint      :/registrati
Methods       : PUT
Input         : json - nome, cognome, username, password
Example       : {"nome": "michela", "cognome": "cognome", "username":"pippo", "password": "123"}       
Output        : Json  - errore, esito
Example       : {"errore": "", "esito": "ok"}, 200
                {"errore": "unique constraint failed: UTENTI.USER", "esito": "ko"}, 404
